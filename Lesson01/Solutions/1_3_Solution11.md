## Problem 1.3.11
__GPA__ is generally used as a __Ratio__ variable.
* Numerical variable
* There is a definite value of zero, meaning no grade points were earned

Note that GPA can also be used as an Ordinal variable
* 3.5-4.0 = A
* 2.5-3.5 = B
* 1.5-2.5 = C
* 0.5-2.5 = D
* 0.0-0.5 = F
